How to enter
--------
1. Fork this repo
  * git clone git://github.com/tropo/ar_drone_contest.git
2. Create branch w/ your email address
  *  git checkout -b john@tropo.com
3. Add you application and commit
  * Submit pull request to tropo
4. World domination
